# EE CDI

*Last update: 2021*

Dependency inject mechanism introduced with EE 6.

Annotations:

* @Inject
* @xxScoped (Request, Session, Application, Conversation)
* @Dependent
* @Named (makes the bean accessible via EL)
* @PostConstruct
* @PreDestroy
